Emotion Based Music Player

Nirma University

Emotion Based Music Player
  • In our daily life, listening to music has become an integral part since inventions like walkmans and mobiles. Still, we haven't really tapped the full potential of benefits of listening to music.

  • What if you could get the perfect music at the perfect moment ? Sounds like a movie scene, but....
    If that is possible, wouldn't we be able to relax better ? sleep better ? cope with different feelings better ?

  • The research done by many scholars says "Yes".
  • So, using this fact as inspiration, me and my project partner created a music player that facilitates this.

  • This music player basically does these three things
    • Take a live snapshot of a person's face (using front camera of laptop or mobile)
    • Utilize the snapshot to ascertain the emotion/mood the person is feeling (using Microsoft's Emotion API)
    • Play music to complement/counteract the mood (using C#)
      For example:
      If you are really excited or happy, music to listen would be something peppy or party music
      OR
      If you are sad, it'd be a good idea to play slow music to relax you and make you feel better

Search Functionality for Huffman Compressed Data

Nirma University

Search Functionality for Huffman Compressed Data
  • This project was undertaken to improve the time required to search files compressed using Huffman encoding.
    Generally, if we need to search one of the multiple compressed files, we need to do the following steps:
    • Unzip the compressed file
    • Open the file in which you need to search
    • Compress all the files back into a single compressed file

  • With the program we created, searching was possible directly in compressed data and thereby, reducing overall time required for searching.

  • Since, Huffman encoding is used in many of the mainstream file compression formats like GZIP, PKZIP and BZIP2 and for image formats such as JPEG and PNG, this project can be applied in a lot of real-world searching problems.
